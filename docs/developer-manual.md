# Developer Manual

## Installation
1. Clone repo
2. Run `npm install`
3. Create `.env.local` with Supabase + Numverify keys

## Running
`npm run dev`

## API Endpoints
- POST /api/validate
- GET /api/searches
- GET /api/stats

## Known Issues
- No auth
- Basic validation only

## Future Work
